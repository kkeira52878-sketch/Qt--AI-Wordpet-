# AI English Learning Desktop Pet - 项目规格书

## 1. 项目概述

### 项目名称
**WordPet** - AI英语单词学习桌宠

### 项目目标
创建一个桌面级AI英语单词学习助手，以桌宠形式呈现，用户在浏览网页时可以通过划词或剪贴板监控将生词添加到词汇库，并通过点击桌宠查看和管理词汇库。

### 核心功能
1. **桌宠形态** - 悬浮在桌面上的可爱宠物形象，始终置顶
2. **划词添加** - 全局热键捕获选中文本，自动查询并添加到词汇库
3. **剪贴板监控** - 自动监测剪贴板内容，识别英文单词
4. **词汇库** - 点击桌宠弹出词汇库窗口，支持搜索、分类、复习
5. **超级模式** - AI驱动的智能复习计划和学习分析

## 2. 技术架构

### 技术栈
- **框架**: Qt 6.x (推荐) 或 Qt 5.x
- **语言**: C++ (推荐) 或 Python (PyQt5/PySide6)
- **UI框架**: Qt Widgets (C++) 或 Qt Quick/QML
- **数据库**: SQLite (本地存储)
- **API**: 有道词典API / Bing Dictionary API

## 3. 功能详细规格

### 3.1 桌宠窗口 (PetWindow)

**窗口属性:**
- 尺寸: 120x120 像素（可自定义）
- 无边框、透明背景
- 始终置顶: `Qt::WindowStaysOnTopHint`
- 可拖拽移动
- 可最小化到系统托盘

**交互行为:**
- 单击: 打开词汇库窗口
- 双击: 打开设置窗口
- 右键: 上下文菜单（退出、设置、关于）
- 拖拽: 移动桌宠位置

**动画状态:**
- 空闲状态: 轻微浮动动画
- 监听状态: 耳朵竖起动画
- 学习状态: 眨眼动画

### 3.2 词汇库窗口 (VocabularyWindow)

**布局:**
```
┌─────────────────────────────────────┐
│ [搜索框] [筛选:全部/已掌握/生词] [排序] │
├─────────────────────────────────────┤
│ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐    │
│ │ 单词 │ │ 单词 │ │ 单词 │ │ 单词 │    │
│ │ 卡片 │ │ 卡片 │ │ 卡片 │ │ 卡片 │    │
│ └─────┘ └─────┘ └─────┘ └─────┘    │
├─────────────────────────────────────┤
│ [分页导航]                          │
└─────────────────────────────────────┘
```

**功能:**
- 单词卡片显示: 单词、音标、释义、添加时间
- 点击卡片: 显示详细信息弹窗
- 搜索: 按单词/释义搜索
- 筛选: 全部/已掌握/生词/收藏
- 排序: 时间/字母/掌握程度
- 删除: 左滑或删除按钮

### 3.3 划词添加系统

**工作流程:**
1. 用户在任意应用选中文本
2. 按全局热键 `Ctrl+Shift+W`
3. 捕获选中文本，发送至词典API
4. 显示查询结果气泡
5. 用户确认后添加到数据库

**技术实现:**
- 全局热键注册: 使用Qt的global event filter
- 选中文本获取: 通过剪贴板或 Accessibility API
- 跨平台兼容: Windows平台特殊处理

### 3.4 剪贴板监控

**功能:**
- 后台监控剪贴板变化
- 识别英文单词（正则: `[a-zA-Z]{2,20}`）
- 自动弹出添加确认气泡
- 可在设置中开启/关闭

### 3.5 词典API集成

**支持的API:**
- 有道词典API (免费配额)
- Bing Dictionary API (备用)

**获取信息:**
- 单词原形
- 音标 (IPA)
- 释义 (简明/详细)
- 例句
- 词根词缀

### 3.6 超级模式 (Super Mode)

**功能:**
- **智能复习**: 基于艾宾浩斯遗忘曲线安排复习
- **学习统计**: 每日/每周学习量图表
- **单词测试**: 拼写测试、选择测试
- **进度追踪**: 掌握度百分比、连续学习天数

**UI:**
- 独立窗口，点击桌宠连续3次激活
- 包含: 复习模块、统计图表、测试入口

### 3.7 云同步 (可选)

**同步内容:**
- 词汇库数据
- 学习进度
- 设置偏好

**技术方案:**
- WebDAV同步 (自建服务器)
- 或 LeanCloud/Supabase BaaS

## 4. 数据模型

### 单词表 (words)
| 字段 | 类型 | 描述 |
|------|------|------|
| id | INTEGER | 主键自增 |
| word | TEXT | 单词原文 |
| phonetic | TEXT | 音标 |
| definition | TEXT | 释义 |
| example | TEXT | 例句 |
| mastery | INTEGER | 掌握程度 (0-5) |
| is_favorite | BOOLEAN | 是否收藏 |
| created_at | DATETIME | 添加时间 |
| last_reviewed | DATETIME | 最后复习时间 |
| next_review | DATETIME | 下次复习时间 |

### 学习记录表 (review_history)
| 字段 | 类型 | 描述 |
|------|------|------|
| id | INTEGER | 主键自增 |
| word_id | INTEGER | 外键 |
| reviewed_at | DATETIME | 复习时间 |
| result | INTEGER | 复习结果 (0-3) |

### 设置表 (settings)
| 字段 | 类型 | 描述 |
|------|------|------|
| key | TEXT | 设置项 |
| value | TEXT | 设置值 |

## 7. 注意事项

1. **平台特殊性**: Windows平台需处理UTF-8编码、全局热键权限
2. **资源限制**: 词典API有请求频率限制，需做缓存
3. **用户体验**: 桌宠应保持低CPU/内存占用
4. **数据安全**: 本地数据库应做基础加密
