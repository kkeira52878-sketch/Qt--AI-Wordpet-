# 英语单词学习 App - 开发进度追踪

## 项目概述
- **项目名称**: 英语单词学习 App
- **技术栈**: Python + PyQt6
- **开发日期**: 2026-05-27

## 功能需求摘要
1. 加载界面 - 显示"请输入文本"，停留1秒
2. 首页 - 4个图标（学习新单词、复习本、更换单词本、个人主页）
3. 学习新单词 - 随机抽取单词，显示中英文、播放发音、例句、图片
4. 复习本 - 显示已加入复习本的单词
5. 更换单词本 - 显示所有单词本，用户选择
6. 个人主页 - 登录账号、切换账号（占位功能）

## 布局要求（已按要求实现）
- 图标4在页面最左上角，比3更靠上，长方形，与3同宽但更短
- 图标3在页面最上方中间，较小，水平的长方形
- 图标1、2最大，一样大，并列排布

## 功能说明

### 加载界面
- 显示"请输入文本"
- 停留1秒后自动跳转首页

### 首页布局
- 图标4（个人主页）：左上角，80x25，紫色
- 图标3（更换单词本）：顶部中间，200x35，红色
- 图标1（学习新单词）：左侧，165x180，蓝色
- 图标2（复习本）：右侧，165x180，绿色

### 学习新单词
- 显示单词英文、音标、中文释义
- 例句在单词下方
- 图片（如果有的话）在单词上方
- 自动播放英文发音（使用gTTS）
- 提供"播放发音"按钮
- Pass按钮：测试用，将单词加入复习本

### 复习本
- 显示学习的单词
- 相同的显示形式（英文、音标、中文、例句、图片）
- Pass按钮：测试用，不加入复习本

### 更换单词本
- 显示所有单词本
- 突出显示当前选中的单词本
- 用户点击即可切换

### 个人主页
- 登录账号（显示"暂未实现"）
- 切换账号（显示"暂未实现"）
